Welcome to Kitty!

This is the web page where we will introduce you into how to run the bot, as well as some other garbage. Also I promise to make this look better.

Kitty is meant to be a sort-of "operational baseline" for Discord bots, off which you can base your bot by adding more modules.

Features I want to put in

  • Stocks
  • Global Chat
  • Server Analytics
  • Compatibility with Python
  • !link - with two options: Automated Setup (!invite auto) and Customized Setup (!invite custom)
    • Automated Setup - This link will include the admin permissions you will give to the bot, no additional roles.
    • Custom Setup - This link will not include any permissions, you can only invite the bot and then give them a specific role set up for members like them.
